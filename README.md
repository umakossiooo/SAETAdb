## BACKEND OF SIPO

# Installations

you need to have typescript (globally) and node installed beforehand

```
npm i

```

# Configure the .env.example

Remember modify the file and delete the .example
it needs to be .env

- PORT = 8080 (example it can be any number)
- DB_NAME=
- DB_USER=
- DB_PASS=
- DB_HOST=
- DB_PORT=

# Compile

```
tsc

```

# Run

```
npm run start

```
